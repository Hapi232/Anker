Introduction

In today’s era of digital transformation, a well-designed website serves as a crucial key by allowing effective interaction between customer and a brand. In this project, we chose the official website of Anker Innovations, a leading global brand that known for its mobile accessories. Anker specializes in power and charging solutions, connectivity and data transfer, and smart home devices. The website that we evaluate is https://www.anker.com.

This evaluation project is to study the effectiveness of website to meets users need in navigating, finding information, and accessing the website features. The goal of this project is to find out the strengths and weaknesses usability and provide guidelines to enhance the overall interface design.

Methodology

We conducted a structured user observation study with 3 participants. They are familiar with online shopping but not Anker’s site. Each participant performed 5 tasks before answering the evaluation form:

  • Find and view details of a power bank
  • Search for USB-C cable
  • Locate contact or feedback
  • Locate warranty or support
  • Switch language or region

Evaluation Results and Design Challenges

(a) Website Usage

Users rated the site 4/5 overall. Some users found the homepage messaging unclear and suggested better content labeling and layout consistency.

(b) Interface Design

The site looks visually appealing. However, one user couldn’t locate the search bar easily. Guidance could be helpful for new users.

(c) Feedback Mechanism

One user struggled to find the contact form. Most preferred a floating button and clearer confirmation after submitting feedback.

4.1 Suggested Guidelines

Homepage Messaging
  • Add tagline
  • Use bold headings
Navigation Bar
  • Improve search visibility
  • Group categories
Floating Feedback
  • Persistent button
  • Expandable form
On-Screen Guidance
  • Tooltips
  • Quick tour option
Confirmation
  • Popup messages ≥ 3s
  • Track request link

Conclusion

Anker’s website is visually strong and generally easy to use. However, users found the homepage message unclear and had issues locating key features. Suggestions include better labels, improved navigation and onboarding support.

Applying these changes would enhance usability and boost user satisfaction, helping them achieve goals confidently.
